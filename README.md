# Paisabazaar-Banking-Fraud-Analysis
Banking fraud analysis using python

## Project Overview
This project analyzes loan applicant data to understand income patterns,
credit scores, and loan approval trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Majority of applicants fall into mid-income range
- Higher credit scores increase loan approval chances
- Certain income groups show higher default risk

## Business Impact
These insights help financial institutions:
- Reduce loan default risk
- Improve approval decisions
- Target the right customers
